# karto-microservice-generator

A CLI tool for generating .NET microservice Clean Architecture project structures.

## Installation

```bash
dotnet tool install --global karto-microservice-generator
```

## Usage

```bash
msgen
```

## Features

- Generate .NET microservice solution structure
- Create BuildingBlocks projects
- Create ApiGateway project
- Create service projects using Clean Architecture
- Add project references automatically
